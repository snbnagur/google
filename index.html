WHAT IS ANDROID

  • It is a open source software platform and operating system for mobile device
  • Based on the Linux kernel
  • Developed by Google and later the Open Handset Alliance (OHA)
  • Allows writing managed code in the Java language
  • Android has its own virtual machine i.e. DVM(Dalvik Virtual Machine), which is used for executing the android applicatn.
  • Google purchased the initial developer of the software, android incorporated in 2005.
  • Open Handset Alliance
  • The open handset alliance(OHA) is a business alliance of firm to develop open standard for mobile devices.
  • Devoted to advancing open standards for mobile devices
  • Develop technologies that will significantly lower the cost of developing and distributing mobile devices and services.
